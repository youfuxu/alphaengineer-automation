# Alpha Engineer — IG Stories 內容策略
建立：2026-06-17

**目的：** IG Stories 補強 carousel 的觸及（Stories 先出現在 feed 頂部），提高互動率和追蹤轉換。

**頻率：** 每週 3-5 個 Stories（carousel 發布日 + 2-3 個平日）

---

## Stories 分類

### Type 1: Quick Poll（快速投票）
最高互動率，Instagram 算法最愛

**Story #1 — Tool poll**
```
背景：深色 #0D1117
文字：Which AI tool are you using right now?
Poll: [Claude] vs [ChatGPT]
```

**Story #2 — Side hustle stage poll**
```
背景：深色
文字：Where are you in your side hustle journey?
Poll: [Just starting] vs [Already earning]
```

**Story #3 — Platform preference**
```
背景：深色
文字：For passive income content, do you prefer:
Poll: [Short videos] vs [Written carousels]
```

---

### Type 2: Micro-tip（30-second insight）
教育性內容，高 save rate

**Story #4 — Weekly Claude prompt**
```
每週三 carousel 發布後發布：
背景：深色
文字：今週 Claude tip:
"Give it role + rules + examples. Never just describe tone."
Text: Save this for your next Claude session.
```

**Story #5 — Quick stat**
```
背景：深色
大字：$500/month
小字：Invested at 25 → $1.13M at 55
底部：Do the math. Start now.
```

**Story #6 — Tool spotlight**
```
背景：深色
圖片：工具截圖
文字：This free tool just saved me 2 hours.
     ElevenLabs free tier → 10,000 chars/month of AI voice
     Perfect for YouTube Shorts narration.
     (Affiliate link in bio)
```

---

### Type 3: BTS Reveal（幕後透明）
建立信任感和真實感

**Story #7 — Process reveal**
```
螢幕錄影（15秒）：
展示 HTML carousel 渲染過程或 GitHub Actions 執行畫面
文字 overlay：This is how my posts are made.
             Zero manual design. 100% automated.
```

**Story #8 — Weekly results**
```
背景：深色
文字：This week's numbers:
     IG reach: [X]
     Threads reach: [X]  
     Email opens: [X]
     Building in public.
```

---

### Type 4: Question Box（問答）
讓 Youfu 收集受眾問題，同時增加互動

**Story #9 — Ask me anything**
```
每月 1 次
文字：Ask me anything about AI side hustles 👇
[Question Box sticker]
回覆作為下期 carousel 素材
```

**Story #10 — This week's topic preview**
```
每週二（carousel 前一天）
文字：Tomorrow's post is about:
     [主題提示，製造懸念]
     The one thing engineers don't know about [主題]
     Drop a 🔥 if you want to be notified.
```

---

## Stories 發布時程

| 星期 | Stories 內容 |
|------|------------|
| 週一 | Quick poll（本週主題相關） |
| 週二 | Preview（下一篇 carousel 預告）|
| 週三 | Carousel 發布日 + Micro-tip |
| 週四 | Tool spotlight 或 BTS |
| 週五 | 回顧本週最高互動貼文，問「你怎麼想？」|

---

## 製作工具

- **設計：** Canva 手機版（Stories 1080×1920）
- **文字：** 白色 Space Grotesk 或 Helvetica Neue
- **背景：** #0D1117 深色（品牌一致）
- **互動貼紙：** Poll / Question Box / Quiz（原生 IG 工具）

---

## 與 Carousel 的協作策略

1. **Carousel 發布日（週三）：**
   - 早上 9am 台灣：發 Story 提示「今天有新貼文」
   - 中午 12pm：Carousel 自動發布（GitHub Actions）
   - 下午 3pm：Swipe-up Story 指向 carousel（「今天發了什麼」）

2. **Carousel 復活技（1-2 週後）：**
   - 「重發」高互動 carousel 的第一張圖作為 Story
   - 文字：「如果你錯過了這篇... swipe up」

3. **Affiliate 轉換路徑：**
   - Story → Tool spotlight → bio link → affiliate landing page
   - 使用「Link in bio」不要在 Story 直接放外部連結（算法懲罰）
